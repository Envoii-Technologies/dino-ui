# Toggle

The Toggle component provides a customizable toggle switch that can be used in your web applications. It allows users to switch between two states and can be configured with various props to suit your needs.

## Examples

### Default

The default Toggle component without any customizations.

<Canvas>
	<Story
		name="Default"
		args={{
			onChange: (checked) => alert(`[NOT IMPLEMENTED] ${checked === true ? "true" : "false"}`)
		}}
	>
		{ Template.bind({}) }
	</Story>
</Canvas>

### Custom Class Name

A Toggle component with a custom CSS class applied for styling.

<Canvas>
  <Story
    name="Custom Class Name"
    args={{
      className: 'custom-toggle',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

### Title and Subtitle

You can add title and subtitle props to add a toggle description on the right side of the component.

<Canvas>
	<Story
		name="Title and Subtitle"
		args={{
			title: 'Toggle Label',
			subtitle: 'Additional information about the toggle.',
		}}
	>
		{ Template.bind({}) }
	</Story>
</Canvas>
