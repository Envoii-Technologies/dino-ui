# ModalInfoWindow

The `ModalInfoWindow` component is a versatile and customizable modal window designed to display informative messages, alerts, or warnings to users in your web application. It is a part of your application's user interface that can be easily incorporated to enhance user interaction and provide important information when needed.

	<Story
		name="Example"
		args={{
				title: "Modal Window",
				body: "This is a modal Window.",
				cancelText: "Cancel",
				onAction: () => alert("[NOT IMPLEMENTED]")
		}}
	>
		{ Template.bind({}) }
	</Story>
  <br/>

## Key Features

- **Customization:** The `ModalInfoWindow` component offers various customization options, including setting the title, type (success, warning, error), icon, and body text to tailor the modal's appearance and content to your specific needs.
- **Action Button:** You can optionally include an action button with custom text and a callback function to allow users to perform actions or take next steps directly from the modal.
- **Accessibility:** It includes keyboard accessibility features, making it easy for users to dismiss the modal by pressing the escape key.
