# Badge

The `Badge` component is a versatile element used for displaying labels, statuses, and other information in your application. It can be customized with different colors, sizes, and styles.

## Examples

### Default

The "Default" example showcases a basic badge with default settings. It displays a small, gray badge with the label "Default Badge." This is a straightforward way to use the `Badge` component when no specific customizations are needed.

<Canvas>
  <Story
    name="Default"
    args={{
      label: 'Default Badge',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

### Custom Badge

The "Custom Badge" example demonstrates how you can customize the appearance of the badge. In this case, we have a larger badge with a green "success" color. The pill style is turned off, and the badge has an outline. A "warning" status dot is also displayed.

<Canvas>
  <Story
    name="Custom Badge"
    args={{
      label: 'Custom Badge',
      size: 'lg',
      color: 'success',
      pill: false,
      outline: true,
      status: 'warning',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>
