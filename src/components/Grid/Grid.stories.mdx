# Grid

The Grid component provides a flexible and responsive grid layout system for structuring content in your React applications. It supports different column layouts and spacing options for various screen sizes.

## Examples

### Default

By default, the Grid component creates a mobile-first layout with 4 columns stacked on top of each other.

## API

### Grid

Die `Grid`-Komponente ist der Container für das Rastersystem.

-   `fluid` (optional): Ein boolscher Wert, der angibt, ob das Grid flüssig (maximale Breite) sein soll.

### Row

Die `Row`-Komponente definiert die Anzahl der Spalten für verschiedene Breakpoints.

-   `columns`: Ein Objekt, das die Anzahl der Spalten für `sm`, `md` und `xl` Breakpoints angibt.

### Column

Die `Column`-Komponente ermöglicht die Einstellung von `span` und `spacing` für eine Spalte.

-   `span` (optional): Die Anzahl der Spalten, die diese Spalte einnehmen soll.
-   `spacing` (optional): Der Abstand (Spacing) zwischen dieser Spalte und ihren Nachbarn.
