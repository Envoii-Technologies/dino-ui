# ButtonGroup

The ButtonGroup component is a versatile tool for organizing and displaying groups of buttons in web applications. It offers features for custom styling, compact mode to save space, and fluid mode for dynamic resizing.

## Key Features

- **Button Organization:** Easily group and organize buttons for a cohesive user interface.
- **Custom Styling:** Apply custom CSS class names to style the ButtonGroup according to your design needs.
- **Compact Mode:** Save screen space by enabling the compact mode.
- **Fluid Mode:** Allow the ButtonGroup to expand and fill available space with the fluid mode.

### Compact Mode

The compact prop is useful when you need to conserve space in your web interface. Setting compact to true will make the ButtonGroup reduce its size, ensuring efficient use of screen real estate.

	<Story
		name="Compact Group"
