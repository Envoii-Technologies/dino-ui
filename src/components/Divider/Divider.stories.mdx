# Divider

The Divider component is a versatile and customizable tool designed to enhance the visual separation of content within your React applications. It serves as a horizontal line that can be easily integrated into your user interfaces to create clear visual distinctions between sections of content.

## Examples

### Default

By default, the Divider will show a single, fixed line between content.

When content is provided as children to the component, the divider will appear on both the left and right sides of the component, rather than being a fixed, permanent line.
